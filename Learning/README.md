### Tech stack

- Bootstrap 5.3.2
- AOS 2.3.4
    - AOS means Animate On Scroll
- Fontawsome 6.5.0
    - Fontawsome is icon library
- Normalize 8.0.1
    - Normalize is CSS library for cross-browser consistency
- Confetti 2.12.0
    - Confetti is library for confetti effect on the website, means celebration effect
- Google Fonts
- Vanilla JS
